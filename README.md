## 📖 About

**Finding Oneself** — экспериментальная RPG, где пони **Трикси** просыпается в шахте без воспоминаний. Ей предстоит заново учиться, сражаться и собирать отряд в опасном Вечнозеленом лесу.

**Главная фишка:** визуал и часть логики создаются с помощью нейросетей. Это эксперимент — насколько ИИ может помочь соло-разработчику.
